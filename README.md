# svelte-speech

This demonstrates using the Web Speech API in a Svelte app.

Get a free Google API key at
<https://developers.google.com/custom-search/v1/introduction#identify_your_application_to_google_with_api_key>.

To run this:

- in a terminal

  - `cd server`
  - `npm install`
  - `npm start`

- in another terminal
  - `npm install`
  - `npm run dev`
